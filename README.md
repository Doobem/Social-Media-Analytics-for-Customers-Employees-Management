# Customer-s-and-Employee-s-Management-in-Mayo-Clinic-and-Cleveland-clinic
## Purpose
The purpose of this study is separated into two sections. The first step is to look at thecustomer/patient review data from social media for Mayo Clinic and Cleveland Clinic. Thesenumbers were scraped and visualised. Employee evaluations, on the other hand, were scrapedand visualised. In general, each clinic's customer and employee management should beunderstood. Furthermore, Twitter data from both Clinics were employed in sentiment analysisto monitor their data.

## Design/methodology/approach
The data science project is mostly used to analyse the study structure. To visualise Yelp,Indeed, and Twitter social media data, different tools API were utilised. The visual data beenutilised will be used to draw conclusions and make recommendations for customer andemployee management for both Clinics in the future.

The Employees management on the otherhand, it implies that the world of health-care staffing is complicated and diverse.
